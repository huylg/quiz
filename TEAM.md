## Agents Team

- Product Manager (Planner): collect requirements, document scope, align
  acceptance criteria, and maintain delivery plan for the real-time quiz.
- Design: research users and competitors, define UX flows, and deliver UI
  specs for quiz join, gameplay, and leaderboard states.
- Backend Engineer: design and build real-time session, scoring, and
  leaderboard services with reliable data flow and scalability in mind.
- Mobile Engineer: build the Flutter client for quiz entry, live gameplay,
  and leaderboard views with responsive updates.
- QA Engineer: write test cases and test suites, validate real-time behavior,
  and verify nonfunctional requirements like performance and reliability.
