# Feature Overview

## Summary

This product enables players to join a live quiz by ID, answer questions in
real time, and see their score and leaderboard position update instantly.

## Core Capabilities

- Join a quiz session using a unique quiz ID.
- Receive real-time score updates after each answer.
- View a live leaderboard with deterministic tie handling.
- Maintain session continuity during reconnects.

## Primary User Flow

1. Enter quiz ID and join session.
2. Answer questions as they appear.
3. View score updates and leaderboard changes.
4. Finish the quiz and see final standings.

## Target Audience

- Students and teams participating in live learning sessions.
- Event attendees in real-time trivia experiences.
