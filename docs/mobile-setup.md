# Mobile App Setup

## Prerequisites

- Flutter SDK installed
- A simulator or device configured

## Run

1. Install dependencies:

```
flutter pub get
```

2. Run the app:

```
flutter run
```

## App Structure

- `lib/main.dart`: Entry point and app shell.
- `lib/routes/`: Routing configuration.
- `lib/screens/`: Join, gameplay, and leaderboard screens.
- `lib/viewmodels/`: View models for state and actions.
- `lib/services/`: API and real-time services.
